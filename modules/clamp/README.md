# clamp() Generator

Generate perfect fluid CSS `clamp()` values for responsive typography and spacing.

## How to use:
1. Enter your **Minimum Size** and **Maximum Size** in pixels.
2. Select the type (**Text** or **Spacing**).
3. The generator will calculate the fluid value based on your configured viewport widths.
4. **Click the result** to copy the code to your clipboard.
5. Use the toggle in the header to customize viewport settings (Min/Max width).
