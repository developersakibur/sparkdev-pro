# WebP Pro

Convert and compress your images to WebP format instantly.

## How to use:
1. Set the **Quality** slider (recommended: 25%-75%).
2. Set the **Max File Size** in KB.
3. **Drag and drop** images into the box or click to select files.
4. Click **Convert & Download**.
5. If multiple images are converted, they will be downloaded as a single `.zip` file.
