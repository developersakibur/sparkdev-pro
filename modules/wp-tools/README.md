# WP Quick Tools

Essential shortcuts and tools for WordPress developers.

## Features:
- **WP-Admin**: Quick access to the WordPress dashboard.
- **Normal/Incognito**: Open the current website in a new tab or incognito window.
- **NC Normal/Incognito**: Visit the site while bypassing cache (adds a unique parameter).
- **Speed/DNS**: Analyze the site with PageSpeed Insights or check DNS records.
- **Freepik Tool**: Special tool to extract high-quality image links from Freepik pages.
